A cross-platform MSNP11 client powered by [msnp11-sdk](https://github.com/campos02/msnp11-sdk) and built with [egui](https://github.com/emilk/egui.git).
* [Crosstalk](https://crosstalk.im) is the default server but that can be changed in Personal Settings

### Installing on Arch Linux
If you wish to install on Arch without using the Flatpak, you can clone this repo (or download the source zip) and run `makepkg -si`.
